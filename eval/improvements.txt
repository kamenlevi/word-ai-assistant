To insert a section break and switch to landscape orientation, use the following code:
```javascript
const section = doc.sections.add();
section.orientation = "landscape";
await context.sync();
```

To set the header, use the following code:
```javascript
const header = doc.sections.getFirst().headers.first;
header.text = "Confidential Draft";
await context.sync();
```

To add page numbers to the footer, use the following code:
```javascript
const footer = doc.sections.getFirst().footers.first;
const para = footer.paragraphs.add();
para.text = "Page ";
para.font.size = 12;
const field = para.fields.add("page");
await context.sync();
```

To insert a citation and bibliography, use the following code:
```javascript
const para = doc.paragraphs.add();
para.text = "According to Smith (2021),...";
const citation = para.citations.add("APA");
citation.source = "Smith, 2021";
const bib = doc.sections.getFirst().footers.first;
const bibPara = bib.paragraphs.add();
bibPara.text = "References: ";
const bibField = bibPara.fields.add("bibliography");
await context.sync();
```

To count the words in a document, use the following code:
```javascript
const body = doc.body;
const text = body.getText();
const wordCount = text.split(" ").length;
return `Your document has ${wordCount} words.`;
```

To get the Flesch reading-ease grade level, use the following code:
```javascript
const body = doc.body;
const text = body.getText();
const sentences = text.split(".");
const sentenceCount = sentences.length;
const words = text.split(" ");
const wordCount = words.length;
const syllables = words.reduce((sum, word) => sum + countSyllables(word), 0);
const flesch = 206.835 - 1.015 * (wordCount / sentenceCount) - 84.6 * (syllables / wordCount);
return `Flesch: ${flesch} (grade ${getGrade(flesch)}), ${wordCount} words across ${sentenceCount} sentences.`;
```
