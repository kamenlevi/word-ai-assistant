To insert a heading, use the following code:
```javascript
const heading = doc.paragraphs.add();
heading.heading = 2;
heading.text = "Methodology";
await context.sync();
```

To make a paragraph bold and 14pt, use the following code:
```javascript
const para = doc.paragraphs.getFirst();
para.font.bold = true;
para.font.size = 14;
await context.sync();
```

To apply a table style, use the following code:
```javascript
const table = doc.tables.getFirst();
table.styleBuiltIn = "Grid Table 4 - Accent 1";
await context.sync();
```

To insert an equation, use the following code:
```javascript
const para = doc.paragraphs.add();
const equation = para.inlinePictures.add();
equation.src = "E = mc^2";
await context.sync();
```

To count words in a document, use the following code:
```javascript
const wordCount = doc.body.text.split(" ").length;
return "Your document has " + wordCount + " words.";
```

To list all headings, use the following code:
```javascript
const headings = [];
for (let i = 0; i < doc.paragraphs.items.length; i++) {
  if (doc.paragraphs.items[i].heading) {
    headings.push(doc.paragraphs.items[i].text);
  }
}
return headings;
```

To insert a block quote, use the following code:
```javascript
const para = doc.paragraphs.add();
para.leftIndent = 36;
para.rightIndent = 36;
para.text = "Premature optimization is the root of all evil.";
await context.sync();
```

To set line spacing, use the following code:
```javascript
for (let i = 0; i < doc.paragraphs.items.length; i++) {
  doc.paragraphs.items[i].spacing.lineSpacing = 1.5;
}
await context.sync();
```

To delete a paragraph, use the following code:
```javascript
try {
  const para = doc.paragraphs.getItem(98);
  para.delete();
  await context.sync();
} catch (error) {
  throw new Error("Paragraph does not exist in the document.");
}
```

To change the title, use the following code:
```javascript
const para = doc.paragraphs.getFirst();
para.text = "Q4 Results";
await context.sync();
```
