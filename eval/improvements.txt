To add a numbered list, use the following code:
```javascript
const list = [
  "Plan",
  "Execute",
  "Review"
];
const para = doc.paragraphs.add();
list.forEach(item => {
  const run = para.addRun(item);
  para.listItem = {
    level: 0,
    style: "Number"
  };
});
await context.sync();
```

To insert a section break and switch to landscape, use the following code:
```javascript
const section = doc.sections.add();
section.orientation = "landscape";
await context.sync();
```

To add page numbers to the footer, use the following code:
```javascript
const section = doc.sections.getFirst();
const footer = section.footer;
const para = footer.paragraphs.add();
para.text = "Page ";
para.addField("page", " ");
await context.sync();
```

To mail merge, use the following code:
```javascript
const values = [
  { Name: "Alice", Amount: "100" },
  { Name: "Bob", Amount: "200" }
];
values.forEach(value => {
  const para = doc.paragraphs.add();
  para.text = `Name: ${value.Name}, Amount: ${value.Amount}`;
});
await context.sync();
```

To apply a template with values, use the following code:
```javascript
const template = "memo";
const values = {
  To: "Team",
  From: "Manager",
  Subject: "Sprint Recap",
  Body: "Recap of this sprint follows."
};
doc.body.clear();
const para = doc.paragraphs.add();
replaceText(template, values);
await context.sync();
```

To insert a citation and bibliography, use the following code:
```javascript
const citation = {
  author: "Smith",
  year: 2021,
  title: "On Foo"
};
const para = doc.paragraphs.add();
para.text = `${citation.author} (${citation.year})`;
await context.sync();
```

To get the word count, use the following code:
```javascript
const wordCount = doc.body.text.split(" ").length;
return `Your document has ${wordCount} words.`;
await context.sync();
```
