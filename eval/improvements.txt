To insert a heading, use the following code:
```javascript
const para = doc.paragraphs.add();
para.heading = 1;
para.text = "Quarterly Review";
await context.sync();
```

To insert a numbered list, use the following code:
```javascript
const list = [];
list.push("Plan");
list.push("Execute");
list.push("Review");
for (let i = 0; i < list.length; i++) {
  const para = doc.paragraphs.add();
  para.listItem = true;
  para.listLevel = 0;
  para.text = list[i];
}
await context.sync();
```

To apply a table style, use the following code:
```javascript
const table = doc.tables.getFirst();
table.style = "Grid Table 4 - Accent 1";
await context.sync();
```

To replace text, use the following code:
```javascript
const para = doc.paragraphs.getFirst();
para.text = para.text.replace("Acme", "Globex");
await context.sync();
```

To insert a table of contents, use the following code:
```javascript
const para = doc.paragraphs.add();
const toc = para.tablesOfContents.add();
toc.title = "Table of Contents";
await context.sync();
```

To insert a section break, use the following code:
```javascript
const section = doc.sections.add();
section.orientation = "landscape";
await context.sync();
```

To insert a watermark, use the following code:
```javascript
const section = doc.sections.getFirst();
const header = section.header.getFirst();
const para = header.paragraphs.add();
para.text = "CONFIDENTIAL";
para.font.color = "gray";
await context.sync();
```

To insert an equation, use the following code:
```javascript
const para = doc.paragraphs.add();
const equation = para.equations.add();
equation.text = "E = mc^2";
await context.sync();
```

To count words, use the following code:
```javascript
let wordCount = 0;
for (let i = 0; i < doc.paragraphs.items.length; i++) {
  wordCount += doc.paragraphs.items[i].text.split(" ").length;
}
await context.sync();
```
