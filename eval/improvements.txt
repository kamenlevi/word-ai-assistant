To insert a bullet list, use the following code:
```javascript
const list = [];
list.push("Apples");
list.push("Bananas");
list.push("Cherries");
const para = doc.paragraphs.add();
para.listItem = { level: 0, startAt: 1 };
list.forEach((item) => {
  const run = para.addRun();
  run.text = item;
  run.font.bold = false;
});
await context.sync();
```

To insert an image, use the following code:
```javascript
const para = doc.paragraphs.add();
const run = para.addRun();
run.addPicture("healthcare-image.png", {
  width: 320,
  height: 240,
});
await context.sync();
```

To add a watermark, use the following code:
```javascript
const section = doc.sections.getFirst();
const header = section.header.getFirst();
const para = header.paragraphs.add();
para.text = "CONFIDENTIAL";
para.alignment = "center";
para.font.color = "gray";
await context.sync();
```

To insert a date picker content control, use the following code:
```javascript
const para = doc.paragraphs.add();
const contentControl = para.contentControls.add(
  "DatePicker",
  { title: "Sign Date", tag: "signDate" }
);
await context.sync();
```

To apply a custom theme, use the following code:
```javascript
const theme = {
  name: "Custom Theme",
  background: "#f2f2f2",
  text: "#333",
  headings: [
    { level: 1, font: "Arial", size: 24, color: "#00698f" },
    { level: 2, font: "Arial", size: 18, color: "#008000" },
  ],
};
doc.theme = theme;
await context.sync();
```

To insert a citation and bibliography, use the following code:
```javascript
const citation = "Smith 2021";
const para = doc.paragraphs.add();
para.citations.add(citation);
const bib = doc.bibliography;
bib.citations.add(citation);
await context.sync();
```
