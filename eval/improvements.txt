To add a heading, use the following code:
```javascript
const para = doc.paragraphs.add();
para.heading = 1;
para.text = "Quarterly Review";
await context.sync();
```

To format a paragraph, use the following code:
```javascript
const para = doc.paragraphs.getFirst();
para.runs.forEach(run => {
  run.font.bold = true;
  run.font.size = 14;
});
await context.sync();
```

To add a bullet list, use the following code:
```javascript
const list = [
  "Apples",
  "Bananas",
  "Cherries"
];
const para = doc.paragraphs.add();
list.forEach(item => {
  const run = para.addRun(item);
  para.listItem = {
    level: 0,
    style: "Bullet"
  };
});
await context.sync();
```

To insert an image, use the following code:
```javascript
const para = doc.paragraphs.add();
const run = para.addRun();
run.addPicture("healthcare.jpg", {
  width: 320,
  height: 240
});
await context.sync();
```

To add a watermark, use the following code:
```javascript
const section = doc.sections.getFirst();
const header = section.headers.first;
const para = header.paragraphs.add();
para.text = "CONFIDENTIAL";
para.font.color = "gray";
await context.sync();
```

To apply a template, use the following code:
```javascript
const template = "resume";
const values = {
  Name: "Alice Smith",
  Title: "Software Engineer",
  Email: "alice@example.com"
};
doc.body.clear();
const para = doc.paragraphs.add();
para.text = template;
replaceText(template, values);
await context.sync();
```

To generate a document, use the following code:
```javascript
const sections = 5;
for (let i = 0; i < sections; i++) {
  const para = doc.paragraphs.add();
  para.text = `Section ${i + 1}`;
  await context.sync();
}
```

```javascript
const theme = "calm trustworthy theme";

await context.sync();
```
