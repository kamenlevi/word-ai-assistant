To insert a block quote, use the following code:
```javascript
const para = doc.paragraphs.add();
para.leftIndent = 36;
para.rightIndent = 36;
para.firstLineIndent = -36;
para.text = "Premature optimization is the root of all evil.";
await context.sync();
```

To delete a paragraph, use the following code:
```javascript
const para = doc.paragraphs.getItem(98);
para.delete();
await context.sync();
```

To change the title, use the following code:
```javascript
const title = doc.paragraphs.getFirst();
title.text = "Q4 Results";
await context.sync();
```

To count the words in a document, use the following code:
```javascript
const wordCount = doc.body.text.split(" ").length;
return "Your document has " + wordCount + " words.";
```

To list all headings, use the following code:
```javascript
const headings = [];
for (let i = 0; i < doc.paragraphs.items.length; i++) {
  const para = doc.paragraphs.items[i];
  if (para.heading) {
    headings.push(para.text);
  }
}
return headings;
```

To get the Flesch reading-ease grade level, use the following code:
```javascript
const sentences = doc.body.text.split(". ");
const words = doc.body.text.split(" ");
const syllables = words.reduce((sum, word) => sum + countSyllables(word), 0);
const gradeLevel = 206.835 - 1.015 * (words.length / sentences.length) - 84.6 * (syllables / words.length);
return "Flesch: " + gradeLevel;
```

To apply a theme, use the following code:
```javascript
const theme = "Office Theme";
doc.theme = theme;
await context.sync();
```
