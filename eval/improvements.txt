To make a paragraph bold and 14pt, use the following code:
```javascript
const para = doc.paragraphs.getItem(0);
para.font.bold = true;
para.font.size = 14;
await context.sync();
```

To add a bullet list, use the following code:
```javascript
const list = [];
list.push("Apples");
list.push("Bananas");
list.push("Cherries");
const para = doc.paragraphs.add();
para.listItem = { level: 0, startAt: 1, style: "bullet" };
list.forEach((item) => {
  const run = para.addRun(item);
});
await context.sync();
```

To add a numbered list, use the following code:
```javascript
const list = [];

```

To replace all 'Acme' with 'Globex', use the following code:
```javascript
const para = doc.paragraphs.getItem(0);
para.text = para.text.replace("Acme", "Globex");
await context.sync();
```

To add page numbers to the footer, right-aligned, use the following code:
```javascript
const section = doc.sections.getFirst();
const footer = section.footer.getFirst();
const para = footer.paragraphs.add();
para.alignment = "right";
para.text = "Page {page}";
await context.sync();
```

To insert an image at the cursor, 320pt wide, use the following code:
```javascript
const image = doc.inlinePictures.add("office-workplace", { width: 320 });
await context.sync();
```

To add a CONFIDENTIAL watermark to every page, use the following code:
```javascript
const section = doc.sections.getFirst();
const header = section.header.getFirst();
const para = header.paragraphs.add();
para.text = "CONFIDENTIAL";
para.font.color = "gray";
para.font.size = 24;
await context.sync();
```
