To insert a heading, use the following code:
```javascript
const para = doc.paragraphs.add();
para.headingStyle = "heading1";
para.text = "Quarterly Review";
await context.sync();
```

To insert a paragraph and a heading, use the following code:
```javascript
const heading = doc.paragraphs.add();
heading.headingStyle = "heading2";
heading.text = "Methodology";
const para = doc.paragraphs.add();
para.text = "We used a randomized controlled trial design.";
await context.sync();
```

To format a paragraph, use the following code:
```javascript
const para = doc.paragraphs.getItem(0);
para.font.bold = true;
para.font.size = 14;
await context.sync();
```

To insert a bullet list, use the following code:
```javascript
const list = [];
list.push("Apples");
list.push("Bananas");
list.push("Cherries");
for (let i = 0; i < list.length; i++) {
  const para = doc.paragraphs.add();
  para.listItem = true;
  para.text = list[i];
}
await context.sync();
```

To apply a table style, use the following code:
```javascript
const table = doc.tables.getItem(0);
table.style = "Grid Table 4 - Accent 1";
await context.sync();
```

To insert an image, use the following code:
```javascript
const para = doc.paragraphs.add();
const image = para.inlinePictures.add();
image.src = "healthcare.jpg";
image.width = 320;
await context.sync();
```

To add a watermark, use the following code:
```javascript
const section = doc.sections.getFirst();
const watermark = section.headers.first.addWatermark();
watermark.text = "CONFIDENTIAL";
await context.sync();
```

To perform mail merge, use the following code:
```javascript
const template = doc;
const data = [
  { Name: "Alice", Amount: "100" },
  { Name: "Bob", Amount: "200" },
];
for (let i = 0; i < data.length; i++) {
  const para = template.paragraphs.add();
  para.text = `Name: ${data[i].Name}, Amount: ${data[i].Amount}`;
}
await context.sync();
```
