To insert a bullet list, use the following code:
```javascript
const list = [];
list.push("Apples");
list.push("Bananas");
list.push("Cherries");
for (let i = 0; i < list.length; i++) {
  const para = doc.paragraphs.add();
  para.listItem = true;
  para.text = list[i];
}
await context.sync();
```

To add page numbers to the footer, use the following code:
```javascript
const section = doc.sections.getFirst();
const footer = section.footer.getFirst();
const para = footer.paragraphs.add();
para.text = "Page {page}";
para.alignment = "right";
await context.sync();
```

To insert an image, use the following code:
```javascript
const para = doc.paragraphs.add();
const image = para.inlinePictures.add();
image.src = "office-workplace.jpg";
image.width = 320;
await context.sync();
```

To apply a template, use the following code:
```javascript
const template = "memo";
const para = doc.paragraphs.add();
para.text = "To: Team\nFrom: Manager\nSubject: Sprint Recap\nBody: Recap of this sprint follows.";
await context.sync();
```

To design a theme, use the following code:
```javascript
const theme = {
  font: "Calibri",
  color: "blue",
  size: 12,
};
for (let i = 0; i < doc.paragraphs.items.length; i++) {
  doc.paragraphs.items[i].font.name = theme.font;
  doc.paragraphs.items[i].font.color = theme.color;
  doc.paragraphs.items[i].font.size = theme.size;
}
await context.sync();
```

To tweak a theme, use the following code:
```javascript
const theme = {
  font: "Calibri",
  color: "blue",
  size: 12,
};

  doc.paragraphs.items[i].font.name = theme.font;
  doc.paragraphs.items[i].font.color = "forestgreen";
}
await context.sync();
```
