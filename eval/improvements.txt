To insert a heading with a specific level, use the following code:
```javascript
const para = doc.paragraphs.add();
para.text = "Methodology";
para.heading = "Heading2";
await context.sync();
```

To add a bullet list, use the following code:
```javascript
const para = doc.paragraphs.add();
const list = para.list;
list.startAt(1);
list.items.add("Apples");
list.items.add("Bananas");
list.items.add("Cherries");
await context.sync();
```

To add a numbered list, use the following code:
```javascript
const para = doc.paragraphs.add();
const list = para.list;
list.startAt(1);
list.items.add("Plan");
list.items.add("Execute");
list.items.add("Review");
await context.sync();
```

To replace text, use the following code:
```javascript
const results = doc.body.search("Acme", { matchCase: true });
results.load("text");
await context.sync();
results.items.forEach(r => { r.text = "Globex"; });
await context.sync();
```

To make text bold and red, use the following code:
```javascript
const results = doc.body.search("WARNING", { matchCase: true });
results.load("font");
await context.sync();
results.items.forEach(r => { r.font.bold = true; r.font.color = "#FF0000"; });
await context.sync();
```

To add a comment, use the following code:
```javascript
const results = doc.body.search("$4.2M", { matchCase: true });
results.load("text");
await context.sync();
results.items[0].comments.add("check this number");
await context.sync();
```

To add page numbers to the footer, use the following code:
```javascript
const section = doc.sections.getFirst();
const footer = section.footer;
const para = footer.paragraphs.add();
para.text = "Page ";
para.addField("page");
await context.sync();
```

To switch to a two-column layout, use the following code:
```javascript
const section = doc.sections.getFirst();
section.columns = 2;
await context.sync();
```

To insert an image at the cursor, use the following code:
```javascript
const para = doc.paragraphs.add();
const run = para.addRun();
run.addPicture("office-workplace.jpg", {
  width: 320
});
await context.sync();
```
