To insert a heading, use the following code:
```javascript
const para = doc.paragraphs.add();
para.heading = 1;
para.text = "Quarterly Review";
await context.sync();
```

To insert a section break, use the following code:
```javascript
const section = doc.sections.add();
section.orientation = "landscape";
await context.sync();
```

To add page numbers, use the following code:
```javascript
const section = doc.sections.getFirst();
const footer = section.footer.getFirst();
const para = footer.paragraphs.add();
para.text = "Page ";
para.alignment = "right";
await context.sync();
```

To insert an image, use the following code:
```javascript
const para = doc.paragraphs.add();
const image = para.inlinePictures.add();
image.width = 320;
await context.sync();
```

To add a watermark, use the following code:
```javascript
const section = doc.sections.getFirst();
const header = section.header.getFirst();
const para = header.paragraphs.add();
para.text = "CONFIDENTIAL";
para.font.color = "gray";
await context.sync();
```

To apply a template, use the following code:
```javascript
const template = "resume";
doc.template = template;
await context.sync();
```

To insert a list, use the following code:
```javascript
const para = doc.paragraphs.add();
para.list = { type: "bullet", items: ["Apples", "Bananas", "Cherries"] };
await context.sync();
```

To insert a footnote, use the following code:
```javascript
const para = doc.paragraphs.add();
para.text = "See appendix A.";
para footnote = true;
await context.sync();
```

To style a table, use the following code:
```javascript
const table = doc.tables.add();
```
