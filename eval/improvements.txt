To insert a heading, use the following code:
```javascript
const para = doc.paragraphs.add();
para.text = "Quarterly Review";
para.heading = "Heading1";
await context.sync();
```

To add a paragraph below a heading, use the following code:
```javascript
const para = doc.paragraphs.add();
para.text = "We used a randomized controlled trial design.";
await context.sync();
```

To apply a table style, use the following code:
```javascript
const table = doc.tables.add();
table.style = "Grid Table 4 - Accent 1";
await context.sync();
```

To set the header, use the following code:
```javascript
const section = doc.sections.getFirst();
const header = section.headers.getFirst();
const para = header.paragraphs.add();
para.text = "Confidential Draft";
await context.sync();
```

To insert an image, use the following code:
```javascript
const para = doc.paragraphs.add();
const run = para.addRun();
run.addPicture("healthcare.jpg", {
  width: 320
});
await context.sync();
```

To add a watermark, use the following code:
```javascript
const section = doc.sections.getFirst();
const header = section.headers.getFirst();
const para = header.paragraphs.add();
para.text = "CONFIDENTIAL";
para.font.color = "gray";
para.font.size = 48;
await context.sync();
```

To apply a template, use the following code:
```javascript
const template = "resume";
doc.template = template;
const fields = {
  Name: "Alice Smith",
  Title: "Software Engineer",
  Email: "alice@example.com"
};
Object.keys(fields).forEach(field => {
  const para = doc.paragraphs.add();
  para.text = `${field}: ${fields[field]}`;
});
await context.sync();
```

To design a theme, use the following code:
```javascript
const theme = {
  name: "calm trustworthy theme for a healthcare startup",
  colors: ["#4567b7", "#6495ed", "#87ceeb"],
  fonts: ["Calibri", "Arial"]
};
doc.theme = theme;
await context.sync();
```

To tweak a theme, use the following code:
```javascript
const theme = doc.theme;
theme.colors = ["#2f4f7f", "#4682b4", "#6495ed"];
theme.fonts = ["Calibri", "Arial"];
doc.theme = theme;
await context.sync();
``
