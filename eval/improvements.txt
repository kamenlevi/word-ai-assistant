To insert a bullet list, use the following code:
```javascript
const para = doc.paragraphs.add();
para.list = { type: "bullet", items: ["Apples", "Bananas", "Cherries"] };
await context.sync();
```

To insert a numbered list, use the following code:
```javascript
const para = doc.paragraphs.add();
para.list = { type: "number", items: ["Plan", "Execute", "Review"] };
await context.sync();
```

To replace text with a regex pattern, use the following code:
```javascript
const regex = /\b[A-Za-z0-9._%+-]+@[A-Za-z0-9.-]+\.[A-Z|a-z]{2,}\b/g;
await replaceText(regex, "[redacted]", { matchCase: false });
await context.sync();
```

To insert a comment on a specific text, use the following code:
```javascript
const para = doc.paragraphs.getFirst();
const range = para.search("$4.2M");
await insertComment(range, "check this number");
await context.sync();
```

To insert a continuous section break and switch to landscape, use the following code:
```javascript
const section = doc.sections.add();
section.orientation = "landscape";
section.sectionBreak = "continuous";
await context.sync();
```

To set the header, use the following code:
```javascript
const section = doc.sections.getFirst();
const header = section.header.getFirst();
const para = header.paragraphs.add();
para.text = "Confidential Draft";
await context.sync();
```

To insert an image at the cursor, use the following code:
```javascript
const para = doc.paragraphs.add();
const image = para.inlinePictures.add();
image.width = 320;
await context.sync();
```

To add a watermark to every page, use the following code:
```javascript
const sections = doc.sections;
sections.load();
await context.sync();
sections.items.forEach((section) => {
  const header = section.header.getFirst();
  const para = header.paragraphs.add();
  para.text = "CONFIDENTIAL";
  para.font.color = "gray";
});
await context.sync();
```

```javascript
const template = "template.docx";
const data = [

  { Name: "Bob", Amount: "200" },

await mailMergeReplace(template, data);
