To center-align a paragraph, use the following code:
```javascript
const para = doc.paragraphs.getFirst();
para.alignment = "center";
await context.sync();
```

To change the font size of a paragraph, use the following code:
```javascript
const para = doc.paragraphs.getFirst();
para.font.size = 28;
await context.sync();
```

To add a bullet list, use the following code:
```javascript
const list = ["Apples", "Bananas", "Cherries"];
for (let i = 0; i < list.length; i++) {
  const para = doc.paragraphs.add();
  para.listItem = true;
  para.text = list[i];
}
await context.sync();
```

To add a numbered list, use the following code:
```javascript
const list = ["Plan", "Execute", "Review"];
for (let i = 0; i < list.length; i++) {
  const para = doc.paragraphs.add();
  para.listItem = true;
  para.text = (i + 1) + ". " + list[i];
}
await context.sync();
```

To add page numbers to the footer, use the following code:
```javascript
const footer = doc.sections.getFirst().footers.first;
const para = footer.paragraphs.add();
para.alignment = "right";
para.text = "Page ";
const field = para.fields.add("page");
await context.sync();
```

To insert an image, use the following code:
```javascript
const para = doc.paragraphs.add();
const image = para.inlinePictures.add();
image.src = "healthcare";
image.width = 320;
await context.sync();
```

To add a watermark, use the following code:
```javascript
const section = doc.sections.getFirst();
const header = section.headers.first;
const para = header.paragraphs.add();
para.text = "CONFIDENTIAL";
para.font.color = "gray";
await context.sync();
```
