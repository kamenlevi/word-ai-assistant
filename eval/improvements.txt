To insert a heading, use the following code:
```javascript
const para = doc.paragraphs.add();
para.text = "Methodology";
para.heading = "Heading2";
await context.sync();
```

To make a paragraph bold and 14pt, use the following code:
```javascript
const para = doc.paragraphs.getItem(0);
para.font.bold = true;
para.font.size = 14;
await context.sync();
```

To add a bullet list, use the following code:
```javascript
const para = doc.paragraphs.add();
const list = para.lists.add();
list.listType = "bullet";
list.addItem("Apples");
list.addItem("Bananas");
list.addItem("Cherries");
await context.sync();
```

To add a numbered list, use the following code:
```javascript
const para = doc.paragraphs.add();
const list = para.lists.add();
list.listType = "number";
list.addItem("Plan");
list.addItem("Execute");
list.addItem("Review");
await context.sync();
```

To insert a Table of Contents, use the following code:
```javascript
const toc = doc.tablesOfContents.add();
toc.title = "Table of Contents";
toc.minLevel = 1;
toc.maxLevel = 3;
await context.sync();
```

To add page numbers to the footer, use the following code:
```javascript
const section = doc.sections.getFirst();
const footer = section.footer;
const para = footer.paragraphs.add();
para.text = "Page ";
para.alignment = "right";
await context.sync();
```

To insert a watermark, use the following code:
```javascript
const section = doc.sections.getFirst();
const header = section.header;
const para = header.paragraphs.add();
para.text = "CONFIDENTIAL";
para.alignment = "center";
await context.sync();
```

To mail merge, use the following code:
```javascript
const template = doc;
const data = [
  { Name: "Alice", Amount: "100" },
  { Name: "Bob", Amount: "200" },
];
data.forEach((item) => {
  const para = template.paragraphs.add();
  para.text = `Name: ${item.Name}, Amount: ${item.Amount}`;
});
await context.sync();
```
