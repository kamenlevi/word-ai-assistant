To insert a heading, use the following code:
```javascript
const para = doc.paragraphs.add();
para.text = "Quarterly Review";
para.heading = "Heading1";
await context.sync();
```

To apply a style to a paragraph, use the following code:
```javascript
const para = doc.paragraphs.getItem(2);
para.style = "Quote";
await context.sync();
```

To insert an image, use the following code:
```javascript
const para = doc.paragraphs.add();
const run = para.addRun();
run.addPicture("healthcare.jpg", {
  width: 320
});
await context.sync();
```

To add a watermark to every page, use the following code:
```javascript
const section = doc.sections.getFirst();
const header = section.header;
const para = header.paragraphs.add();
para.text = "CONFIDENTIAL";
para.alignment = "center";
await context.sync();
```

To mail merge, use the following code:
```javascript
const template = doc;
const data = [
  { Name: "Alice", Amount: "100" },
  { Name: "Bob", Amount: "200" },
];
data.forEach((item) => {
  const para = template.paragraphs.add();
  para.text = `Name: ${item.Name}, Amount: ${item.Amount}`;
});
await context.sync();
```

```javascript
const template = "resume";
```
