To insert a heading, use the following code:
```javascript
const para = doc.paragraphs.add();
para.text = "Methodology";
para.heading = "Heading2";
await context.sync();
```

To add a bullet list, use the following code:
```javascript
const list = [
  "Apples",
  "Bananas",
  "Cherries"
];
const para = doc.paragraphs.add();
list.forEach(item => {
  const run = para.addRun(item);
  para.listItem = {
    level: 0,
    style: "Bullet"
  };
});
await context.sync();
```

To get the readability score, use the following code:
```javascript
const text = doc.body.text;
const sentences = text.split(".").length;
const words = text.split(" ").length;
const score = 206.835 - 1.015 * (words / sentences) - 84.6 * (syllables / words);
return `Readability score: ${score}`;
await context.sync();
```

To list all headings, use the following code:
```javascript
const headings = [];
doc.paragraphs.items.forEach(para => {
  if (para.heading) {
    headings.push(para.text);
  }
});
return headings;
await context.sync();
```
