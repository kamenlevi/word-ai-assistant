To center-align a paragraph, use the following code:
```javascript
const para = doc.paragraphs.getItem(0);
para.alignment = "center";
await context.sync();
```

To set line spacing to 1.5 on every paragraph, use the following code:
```javascript
const paras = doc.paragraphs;
for (let i = 0; i < paras.items.length; i++) {
  paras.items[i].paragraphFormat.lineSpacing = 1.5;
}
await context.sync();
```

To delete a paragraph, use the following code:
```javascript
const para = doc.paragraphs.getItem(99);
para.delete();
await context.sync();
```

To change the title, use the following code:
```javascript
const para = doc.paragraphs.getItem(0);
para.text = "Q4 Results";
await context.sync();
```

To insert a block quote, use the following code:
```javascript
const para = doc.paragraphs.add();
para.leftIndent = 36;
para.firstLineIndent = -36;
para.text = "Premature optimization is the root of all evil.";
await context.sync();
```

To switch the page orientation to landscape, use the following code:
```javascript
const section = doc.sections.getFirst();
section.pageOrientation = "landscape";
await context.sync();
```

To get the word count, use the following code:
```javascript
const wordCount = doc.body.text.split(" ").length;
return "Your document has " + wordCount + " words.";
```

To get the Flesch reading-ease grade level, use the following code:
```javascript
const sentences = doc.body.text.split(". ");
const words = doc.body.text.split(" ");
const syllables = words.reduce((sum, word) => sum + getSyllableCount(word), 0);
const flesch = 206.835 - 1.015 * (words.length / sentences.length) - 84.6 * (syllables / words.length);
return "Flesch: " + flesch;
```

To list all headings, use the following code:
```javascript
const headings = [];
for (let i = 0; i < doc.paragraphs.items.length; i++) {
  if (doc.paragraphs.items[i].headingStyle) {
    headings.push(doc.paragraphs.items[i].text);
  }
}
return headings;
```
