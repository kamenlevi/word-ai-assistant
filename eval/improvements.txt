To add a Heading 1, use the following code:
```javascript
const para = doc.paragraphs.add();
para.text = "Quarterly Review";
para.heading = 1;
await context.sync();
```

To add a bullet list, use the following code:
```javascript
const list = [];
list.push("Apples");
list.push("Bananas");
list.push("Cherries");
const para = doc.paragraphs.add();
para.listItem = true;
para.list = { type: "bullet" };
list.forEach(item => {
  const run = para.addRun(item);
});
await context.sync();
```

To add a numbered list, use the following code:
```javascript
const list = [];
list.push("Plan");
list.push("Execute");
list.push("Review");
const para = doc.paragraphs.add();
para.listItem = true;
para.list = { type: "number" };
list.forEach(item => {
  const run = para.addRun(item);
});
await context.sync();
```

To add page numbers to the footer, use the existing code.

To insert an image, use the following code:
```javascript
const para = doc.paragraphs.add();
const run = para.addRun();
run.addPicture("office-workplace.jpg", { width: 320 });
await context.sync();
```

To add a watermark, use the following code:
```javascript
const section = doc.sections.getFirst();
const header = section.header.getFirst();
const para = header.paragraphs.add();
para.text = "CONFIDENTIAL";
para.font.color = "#CCCCCC";
para.font.size = 48;
await context.sync();
```

To mail merge, use the following code:
```javascript
const template = doc;
const data = [
  { Name: "Alice", Amount: "100" },
  { Name: "Bob", Amount: "200" },
];
data.forEach(item => {
  const para = template.paragraphs.add();
  para.text = `Name: ${item.Name}, Amount: ${item.Amount}`;
});
await context.sync();
```

To apply a theme, note that this is a complex task and not directly available through the Word Office.js API.

const para = doc.paragraphs.add();
para.text = "(Smith, 2021)";
