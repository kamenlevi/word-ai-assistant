To find and replace text, use the following code:
```javascript
const results = doc.body.search("WARNING", { matchCase: true });
results.load("font");
await context.sync();
results.items.forEach(r => { r.font.bold = true; r.font.color = "#FF0000"; });
await context.sync();
```

To add page numbers to the footer, use the following code:
```javascript
const section = doc.sections.getFirst();
const footer = section.footer.getFirst();
const para = footer.paragraphs.add();
para.text = "Page {page}";
para.alignment = "right";
await context.sync();
```

```javascript

}
```

To list all headings, use the following code:
```javascript
const headings = [];
doc.paragraphs.load("text, level");
await context.sync();
doc.paragraphs.items.forEach((para) => {
  if (para.level > 0) {
    headings.push(para.text);
  }
});
return headings;
```
