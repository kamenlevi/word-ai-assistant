To add a bullet list, use the following code:
```javascript
const para = doc.paragraphs.add();
para.listItem = {
  level: 0,
  alignment: "left",
  startAt: 1,
  format: "bullet"
};
para.text = "Apples, Bananas, Cherries";
await context.sync();
```

To add a numbered list, use the following code:
```javascript
const para = doc.paragraphs.add();
para.listItem = {
  level: 0,
  alignment: "left",
  startAt: 1,
  format: "number"
};
para.text = "Plan, Execute, Review";
await context.sync();
```

To apply a style to a specific paragraph, use the following code:
```javascript
const para = doc.paragraphs.getItem(2);
para.style = "Quote";
await context.sync();
```

To set the header, use the following code:
```javascript
const section = doc.sections.getFirst();
const header = section.headers.getFirst();
header.paragraphs.getFirst().text = "Confidential Draft";
await context.sync();
```

To add page numbers to the footer, use the following code:
```javascript
const section = doc.sections.getFirst();
const footer = section.headers.getLast();
const para = footer.paragraphs.add();
para.alignment = "right";
para.text = "Page {page}";
await context.sync();
```

To add a watermark, use the following code:
```javascript
const section = doc.sections.getFirst();
const header = section.headers.getFirst();
const shape = header.insertShape("text");
shape.text = "CONFIDENTIAL";
shape.rotation = 45;
shape.width = 300;
shape.height = 100;
await context.sync();
```

To apply a template, use the following code:
```javascript
await doc.templates.load("memo");
const template = doc.templates.get("memo");
template.fields.forEach((field) => {
  if (field.label === "To") field.text = "Team";
  if (field.label === "From") field.text = "Manager";
  if (field.label === "Subject") field.text = "Sprint Recap";
  if (field.label === "Body") field.text = "Recap of this sprint follows.";
});
await context.sync();
```

To apply a theme, use the following code:
```javascript
doc.theme = "Office";
await context.sync();
```
