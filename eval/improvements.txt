To insert a heading, use the following code:
```javascript
const para = doc.paragraphs.add();
para.text = "Quarterly Review";
para.style = "Heading 1";
await context.sync();
```

To insert a block quote, use the following code:
```javascript
const para = doc.paragraphs.add();
para.style = "Quote";
para.text = "Premature optimization is the root of all evil.";
await context.sync();
```

To set line spacing, use the following code:
```javascript
const para = doc.paragraphs.getItem(0);
para.paragraphFormat.lineSpacing = 1.5;
para.paragraphFormat.lineSpacingRule = "atLeast";
await context.sync();
```

To delete a paragraph, use the following code:
```javascript
try {
  const para = doc.paragraphs.getItem(99);
  para.delete();
  await context.sync();
} catch (error) {
  console.log("Paragraph 99 does not exist in the document.");
}
```

To change the title, use the following code:
```javascript
const para = doc.paragraphs.getItem(0);
para.text = "Q4 Results";
await context.sync();
```

To count words, use the following code:
```javascript
const wordCount = doc.body.text.split(" ").length;
return "Your document has " + wordCount + " words.";
```

To get readability, use the following code:
```javascript
const stats = await doc.statistics;
const readability = stats.readability;
return "Flesch: " + readability.flesch + " (grade " + readability.grade + "), " + stats.wordCount + " words across " + stats.sentenceCount + " sentences.";
```

To list headings, use the following code:
```javascript
const headings = [];
doc.paragraphs.forEach((para) => {
  if (para.style === "Heading 1" || para.style === "Heading 2" || para.style === "Heading 3") {
    headings.push(para.text);
  }
});
return headings;
```

To switch page orientation, use the following code:
```javascript
const section = doc.sections.getFirst();
section.orientation = "landscape";
await context.sync();
```
