To add a heading, use the following code:
```javascript
const heading = doc.paragraphs.add();
heading.styleBuiltIn = "Heading 1";
heading.text = "Quarterly Review";
await context.sync();
```

To add a bullet list, use the following code:
```javascript
const list = [];
list.push("Apples");
list.push("Bananas");
list.push("Cherries");
const para = doc.paragraphs.add();
para.listItem = true;
para.listFormat.listType = "bullet";
list.forEach(item => {
  const run = para.addRun();
  run.text = item;
  run.font.size = 12;
  para.addBreak();
});
await context.sync();
```

To add page numbers to the footer, use the following code:
```javascript
const section = doc.sections.getFirst();
const footer = section.footer.getFirst();
const para = footer.paragraphs.add();
para.alignment = "right";
para.text = "Page " + section.pageNumbers;
await context.sync();
```

To insert an image, use the following code:
```javascript
const para = doc.paragraphs.add();
const run = para.addRun();
run.addPicture("office-workplace", 320, 200);
await context.sync();
```

To mail merge, use the following code:
```javascript
const data = [
  { Name: "Alice", Amount: "100" },
  { Name: "Bob", Amount: "200" },
];
data.forEach(item => {
  const para = doc.paragraphs.add();
  para.text = `Name: ${item.Name}, Amount: ${item.Amount}`;
});
await context.sync();
```
