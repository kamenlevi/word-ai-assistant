To insert a Heading 2, use the following code:
```javascript
const heading = doc.paragraphs.add();
heading.heading = { level: 2 };
heading.text = "Methodology";
await context.sync();
```

For adding a paragraph, utilize the following code:
```javascript
const para = doc.paragraphs.add();
para.text = "We used a randomized controlled trial design.";
await context.sync();
```

To make a paragraph bold and 14pt, use the following code:
```javascript
const para = doc.paragraphs.getFirst();
para.font.bold = true;
para.font.size = 14;
await context.sync();
```

For adding a bullet list, use the following code:
```javascript
const list = [
  "Apples",
  "Bananas",
  "Cherries"
];
const para = doc.paragraphs.add();
list.forEach(item => {
  const run = para.addRun(item);
  run.font.size = 24;
  para.listItem = {
    level: 0,
    startAt: 1,
    style: "Bullet"
  };
});
await context.sync();
```

To apply the 'Grid Table 4 - Accent 1' style to the existing table, use the following code:
```javascript
const table = doc.tables.getFirst();
table.style = "Grid Table 4 - Accent 1";
await context.sync();
```

For turning on track changes, use the following code:
```javascript
doc.settings.trackChanges = true;
await context.sync();
```

To insert an image, use the following code:
```javascript
const image = doc.inlinePictures.add();
image.src = "healthcare.jpg";
image.width = 320;
await context.sync();
```

For adding a watermark, use the following code:
```javascript
const section = doc.sections.getFirst();
const watermark = section.headers.first.addParagraph();
watermark.text = "CONFIDENTIAL";
watermark.font.color = "gray";
await context.sync();
```

To mail merge, use the following code:
```javascript
const data = [
  { Name: "Alice", Amount: "100" },
  { Name: "Bob", Amount: "200" }
];
data.forEach(item => {
  const para = doc.paragraphs.add();
  para.text = `Name: ${item.Name}, Amount: ${item.Amount}`;
});
await context.sync();
```

To apply a theme, use the following code:
```javascript
doc.theme = "Office Theme";
await context.sync();
``
