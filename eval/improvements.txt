To insert a heading, use the following code:
```javascript
const para = doc.paragraphs.add();
para.headingStyle = "Heading 2";
para.text = "Methodology";
await context.sync();
```

To format a paragraph, use the following code:
```javascript
const para = doc.paragraphs.getFirst();
para.font.bold = true;
para.font.size = 14;
await context.sync();
```

To add a bullet list, use the following code:
```javascript
const list = [];
list.push("Apples");
list.push("Bananas");
list.push("Cherries");
for (let i = 0; i < list.length; i++) {
  const para = doc.paragraphs.add();
  para.listItem = true;
  para.text = list[i];
  para.font.size = 12;
}
await context.sync();
```

To add a numbered list, use the following code:
```javascript
const list = [];
list.push("Plan");
list.push("Execute");
list.push("Review");
for (let i = 0; i < list.length; i++) {
  const para = doc.paragraphs.add();
  para.listItem = true;
  para.text = (i + 1) + ". " + list[i];
  para.font.size = 12;
}
await context.sync();
```

To apply a table style, use the following code:
```javascript
const table = doc.tables.add();
table.style = "Grid Table 4 - Accent 1";
await context.sync();
```

To add page numbers to the footer, use the following code:
```javascript
const footer = doc.sections.getFirst().footers.first;
const para = footer.paragraphs.add();
para.alignment = "right";
para.text = "Page ";
const field = para.fields.add("page");
await context.sync();
```

To insert an image, use the following code:
```javascript
const para = doc.paragraphs.add();
const image = para.inlinePictures.add();
image.src = "office-workplace";
image.width = 320;
await context.sync();
```

To add a watermark, use the following code:
```javascript
const section = doc.sections.getFirst();
const header = section.headers.first;
const para = header.paragraphs.add();
para.text = "CONFIDENTIAL";
para.font.color = "gray";
await context.sync();
```
